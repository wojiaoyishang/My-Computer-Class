# My Computer Class

#### Description
无聊的信息课上，总想出点乐子。多想在屏幕广播时退出玩游戏，多想摆脱老师的控制出尽风头。初中阶段，我制作了幻灭极域，用于摆脱极域的控制。在老师得知后使我几乎在上计算机课时全课都被盯着，要求下架删除。现在......我毕业了！再也不是以前懵懂的初中生了，学会了代码规范，重新审视了审美！我重新制作了My Computer Class软件，用于摆脱老师控制！而这次——支持极域与红蜘蛛！
